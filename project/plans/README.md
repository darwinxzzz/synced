# Plans – Map to Project Directory

This directory holds the map to the project structure and active plan documents. Completed plans are filed under `project/plans-completed/<category>/` by default, or under `project/changelog/plans/` when explicitly requested.

## Project directory map

- `project/KIV/` — Keep in view / backlog
- `project/research/` — Research and discovery artifacts
- `project/build/` — Build artifacts; active plans may also live here
- `project/plans/` — Active plans, `README.md`, and `TODO.md`
- `project/plans-completed/` — Categorized completed-plan archive
- `project/changelog/` — Change entries and the changelog index
- `project/troubleshooting/` — Troubleshooting entries and index

See `TODO.md` for current tasks and `project/plans-completed/README.md` for filing rules.
