## Project Info

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **API:** tRPC v11
- **Database:** Supabase (Postgres + RLS)
- **UI:** Tailwind v4 + Radix UI
- **Testing:** Vitest + Playwright
- **Package Manager:** pnpm
